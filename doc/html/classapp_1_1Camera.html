Detailed Description

Класс камеры

Author
KiK0S
Version
1.0
Date
May 14 2021

Класс Camera отвечает за то, чтобы переводить объекты из системы координат мира в систему координат экрана. Для этого этот объект генерирует матрицу преобразования ии применяет ее к точкам, которые надо отобразить. Камеру можно поворачивать и двигать, что отражается на матрице преобразования.
