Detailed Description

Класс инстанса графического приложения.

Author
KiK0S
Version
1.0
Date
May 14 2021

Класс создает окно размера 1000x1000, в котором будут отображаться объекты. После вызова метода run() приложение переходит в интерактивный режим и реагирует на нажатие на клавиши. Для обработки кадра пользоваться update_and_draw_frame()
